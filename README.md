# Portfolio Tracking Application - BackEnd System

This application is a mock for portfolio tracking in trading systems. Various actions can be performed like SELL, BUY, List your Holdings, list all transactions, etc.

### Actions that can be performed in this application
    
    - Add new security 
    - Get all securities
    - Get security by symbol
    - Get LTP of a security (Latest transaction price)

    - Add new trade - (Buy/Sell)
    - Update a trade
    - Delete a trade (Only Sell type trades can be deleted)

    - Get holdings of a user
    - All trade data of a user
    - Get returns of a user

    - Create a user
    - Get user by email
    - Get all users
